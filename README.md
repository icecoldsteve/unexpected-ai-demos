# Unexpected AI • Profit Automations

Interactive Progressive Web App (PWA) showcasing **5 unexpected AI automations** that autonomously find customers, generate photorealistic previews, calculate ROI, and prepare ready-to-use sales pitches.

Inspired by the viral thread from [@rostikdeni](https://x.com/rostikdeni/status/2065476430688133302).

## 🚀 Live Demo

**https://unexpected-ai-demos.vercel.app** (deployed on Vercel)

## ✨ The 5 Demos

| Demo | What it does | Key Benefit |
|------|--------------|-------------|
| 🏊 **Pool Scanner** (OpenClaw) | Finds homes with large backyards but no pool using public data + satellite. Generates before/after visualizations + value uplift estimates + personalized sales pitch. | Turn cold properties into warm, high-intent leads in seconds |
| 💬 **AI Receptionist** | 24/7 automated replies on WhatsApp/IG/Telegram. Qualifies leads, books appointments, and pushes qualified leads to the owner. | Never lose a customer to slow response again |
| 🏠 **Property Scanner** | Scans entire neighborhoods via satellite + vision AI. Flags roof issues, solar opportunities, insurance risks. Auto-generates offers. | 15–25 warm leads/week with zero manual prospecting |
| 🔍 **AI Visibility Scanner** | Audits websites for ChatGPT/Perplexity/Gemini readability. Generates missing FAQs, optimized content, and schema. | 3–7× increase in AI-driven organic traffic |
| 🎮 **AI Mini Games** | Instantly creates fully branded, playable mini-games (Wheel of Fortune, quizzes) reskinned with client assets. | Sell $3k–$8k interactive experiences that outperform static ads |

## 🛠 Tech Stack

- **Vite + React 19 + TypeScript**
- **Tailwind CSS** (via CDN for instant beautiful styling)
- **Fully client-side** — no backend or API keys required
- **PWA ready** — installable, works offline after first load, add-to-home-screen prompt
- Deploy-ready for **Vercel**

## 🧪 Running Locally

```bash
npm install
npm run dev
```

Then open http://localhost:5173

## 📦 Build & Deploy

```bash
npm run build
```

The `dist/` folder is ready for any static host (Vercel, Netlify, Cloudflare Pages, GitHub Pages).

## 📱 PWA Features

- Add to Home Screen prompt
- Standalone app experience
- Fast loading with cached assets
- Works great on mobile
